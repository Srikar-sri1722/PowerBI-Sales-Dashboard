# PowerBI-Sales-Dashboard
ElectroHub Sales Analysis Dashboard built using Power BI. This project analyzes sales, profit, quantity sold, discounts, and orders across multiple product categories. It includes insights on top/bottom products, sales trends over time, sales vs profit relationships, city-wise performance, and interactive filters for dynamic business analysis.

This project analyzes sales performance using Power BI.  
It provides insights into sales trends, profit analysis, discounts, and product performance.

## Features
- Top and bottom products by sales and profit
- Sales trend analysis
- Sales vs Profit relationship
- City-wise sales analysis
- Interactive filters

## Tools Used
- Power BI
- Excel
